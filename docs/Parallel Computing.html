Testing with bigger chunks

clusts_2 <- makeCluster(3)
registerDoParallel(clusts_2)

system.time({
boot_result <- foreach(i = 1:10,.combine = c, .packages = 'MASS') %dopar% {
  replicate(100,
  median(sample(galaxies, size = length(galaxies), replace = TRUE))
)}

})
   user  system elapsed 
   0.01    0.00    0.31 
stopCluster(clusts_2)

Increasing the size of the chunks reduced the overheads, making it more efficient compared to running individual bootstrap iterations.
